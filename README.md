SkyCast is a modern, responsive weather application built with **React.js**. It provides real-time weather data, supports multiple languages, and features a dynamic UI that adapts to the time of day.

---

## üöÄ Key Features

* **Real-time Data:** Fetched from [OpenWeatherMap API](https://openweathermap.org/).
* **Multilingual:** Full support for **Arabic** and **English** using `i18next`.
* **Dynamic Visuals:** Automatic switching between **Day/Night modes** based on local time.
* **Responsive UI:** Crafted with **Material UI (MUI)** for a seamless experience on all devices.
* **Accurate Timing:** Real-time clock and date formatting using `Moment.js`.

## ‚öôÔ∏è Installation & Setup

Follow these steps to run the project locally:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/elnakhlawi/weather-cast-weather-app.git](https://github.com/elnakhlawi/weather-cast-weather-app.git)
    ```
2.  **Navigate to Project Directory:**
    ```bash
    cd weather-app-react
    ```
3.  **Install Dependencies:**
    ```bash
    npm install
    ```
4.  **Add your API Key:**
    Open `App.jsx` and replace the API key with your own from OpenWeatherMap.
5.  **Start Development Server:**
    ```bash
    npm run dev
    ```
